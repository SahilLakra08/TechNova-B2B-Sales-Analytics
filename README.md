# TechNova Electronics — B2B Sales Analytics

## Project Overview
End-to-end Business Intelligence project simulating real company data 
from Salesforce CRM, SAP ERP, and manual Excel reports.

## Dataset
- 67,000 rows across 4 sheets
- 25 columns covering orders, customers, products, and targets
- Intentionally messy — mixed dates, duplicates, wrong calculations

## What I Built
- Data Quality Report — 12 issues documented and resolved
- 14 Pivot Table analyses across 4 sheets
- Interactive Executive Dashboard

## Key Insights
1. Revenue declined -5.21% in 2024 then recovered +3.31% in 2025
2. Mobile category: #2 revenue but worst margin at 21%
3. Desktop: best category — high volume + high margin
4. North vs South — revenue vs efficiency gap
5. Discount policy uniform — margin gaps driven by product mix

## Tools Used
Microsoft Excel · Power Query · Pivot Tables · PivotCharts
